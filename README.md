# aliyun-cdn-cert-bot
Update your Aliyun's CDN certificate automatically. 自动更新你的阿里云 CDN 证书。

再也不需要安装繁重的 Python。使用轻巧的 Golang 更新你的阿里云 CDN 证书，可以结合 acme.sh、let's encrypt 免费使用证书。
